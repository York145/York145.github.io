get_shell

py@ubuntu:~$ cd Desktop/123
py@ubuntu:~/Desktop/123$ checksec web_shell 
[*] '/home/py/Desktop/123/web_shell'
    Arch:     amd64-64-little
    RELRO:    Partial RELRO
    Stack:    No canary found
    NX:       NX enabled
    PIE:      No PIE (0x400000)
py@ubuntu:~/Desktop/123$

发现是64位有nx保护。先放到IDA pro进行反编译。

  int __cdecl main(int argc, const char **argv, const char **envp)
  {
    puts("OK,this time we will get a shell.");
    system("/bin/sh");
    return 0;
  }

好吧，看来是个入门题。根本没有7颗星的难度。
直接nc 连接即可。
